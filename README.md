**Traffic Fine Management System (C++)**

**Author:**
**Kazi Md. Borhan Uddin (Mahi)**

**Overview:**

This project is a console-based **Traffic Fine Management System** developed in **C++**. It helps traffic authorities manage violation records and allows offenders to view and pay fines. The system uses **file handling** to store data permanently.

 **Main Features:**
- Secure admin login  
- Issue and manage traffic fine records  
- View all cases and search by driving license 
- Dashboard showing total collected and pending fines  
- Fine payment using registered mobile number 
- Transaction ID recorded after payment  
- Data saved in `fines.txt` for future access

**Technical Details:**
- Built using **Object-Oriented Programming**  
- Uses **abstraction, inheritance, and polymorphism**  
- **Friend function** used for file writing  
- Standard **C++ libraries** for file I/O and formatting
